# User Stories (Inferred)

*Inferred from module structure and plan. Suggest review/expansion by product owner.*

## Systems/Experiences

- As a user, I can search (e.g. "hiking egypt") to find relevant experiences
- As a user, I can browse the feed of entries
- As a user, I can vote/like entries
- As a user, I can create and edit my own entries
- As a user, I can use a public entry as a template ("Copy to my systems")
- As a user, I can set visibility per entry (Personal / Unlisted / Public)

## Notes

- As a user, I can create notes using the shared editor
- As a user, I can list, edit, and delete notes
- As a user, I can search across all notes
- As a user, I can organize notes with tags and links

## Journal

- As a user, I can create journal entries
- As a user, I can list journal entries
- As a user, I can open and edit entries
- As a user, I can use guided prompts (morning/evening)
- As a user, I can add mood/theme tags for pattern discovery

## Editor

- As a user, I can create documents
- As a user, I can edit blocks (text, media, checklist, etc.)
- As a user, I can add media with preview
- As a user, I can see live preview while editing
- As a user, I can export to Markdown, HTML, JSON
- As a user, I can search within documents

## People

- As a user, I can add a person
- As a user, I can log conversation notes
- As a user, I can add a birthday and enable BD notifications
- As a user, I can add work context (title, background, last convo)
- As a user, I can view activity timeline per person

## Media

- As a user, I can browse all media in a dedicated tab
- As a user, I can categorize, sort, and search media
- As a user, I can use media picker when adding media in editor

## Auth (Web)

- As a user, I can register with email and password
- As a user, I can log in
- As a user, I can verify my email
- As a user, I can reset/change my password (forgot password flow)
