# Project Idea

## Core Concept

**Systems** is a platform that helps people build and reuse personal or shared "systems" to **minimize cognitive load and stress**.

The philosophy: **externalize don't memorize**, **templates over blank slate**, **reduce decision fatigue**, **design for endurance**. See [systems-philosophy.md](systems-philosophy.md) for details.

## Core Value: Searchable, Reusable Experiences

### Example: Hiking in Egypt (first time)

- User doesn't know what to pack.
- In Systems, they search **"hiking egypt"**.
- They get a **feed/list of experiences** from others (or themselves): what to take, what's needed vs what's not.
- Entries are sorted by votes/likes.
- Alternatively: the user went hiking before and recorded what they took, what they actually needed, and what they didn't need.
- Next time, they (or others) search and find that entry.
- Entries are well-organized, Markdown-formatted, and editable in the editor.

## Content Types

| Feature | Purpose |
|---------|---------|
| **Systems/Experiences Feed** | Searchable repository of reusable guides, packing lists, checklists. Community or personal. Sorted by votes/likes. |
| **Notes** | Personal notes using the shared editor. |
| **Journal** | Journal entries using the shared editor. |
| **People** | Log things about people you know or meet. Conversation notes, birthdays (with optional BD notifications), work context (title, background, notes from last convo). Editor used for notes. |
| **Media** | Dedicated tab showing all media in a categorized way, sorted and searchable. |
| **Editor** | Modular, reusable rich editor. Full Markdown, live preview (Confluence/MarkText style), media with preview, checklists. Used in Notes, Journal, People, and experience entries. |

## Editor Requirements

- **State-of-the-art editing** with live preview (like Confluence or MarkText).
- Full Markdown support.
- Media with preview.
- Checklists.
- Modular design so it can be reused in Notes, Journal, People, and other apps.

## People Feature

- Conversation notes to build on when you meet someone again.
- Birthday logging with optional birthday notifications (per person).
- Work contacts: title, background, notes from last conversation.
- Editor used for all notes about people.

## Inspiration

- **Confluence** – Live preview, structured documentation
- **MarkText** – Markdown editing, split view
- **Day One** – Journaling, media handling
- **Joplin** – Open-source, markdown support
- **Notion** – Block-based editing, flexibility
