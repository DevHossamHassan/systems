# Journal Feature

## Olly Jay: Mental Clarity Practice

*Source: [How Journaling for 10 Minutes a Day Can Transform Your Mental Health](https://medium.com/change-your-mind/how-journaling-for-10-minutes-a-day-can-transform-your-mental-health-b26413406ff2)*

**Core insight**: Journaling is a **mental clarity practice**, not just "emotional" writing. It unloads noise, clarifies priorities, regulates emotions, and reveals patterns. 5–10 min/day can sharpen decision-making (activates prefrontal cortex).

## Features

| Improvement | Description |
|-------------|-------------|
| **10-min sessions** | Low-friction daily target; optional gentle reminder. Consistency > length. |
| **Morning vs evening** | Morning = planning, priorities; Evening = reflection, processing. Different templates/prompts. |
| **Guided prompts** | Built-in prompts: "What outcome actually matters this week?", "Where did I lead well today?", "What's draining me?", "What patterns am I noticing?" |
| **Brain dump mode** | Unfiltered free-write to clear mental clutter; weekly or ad-hoc entry type. |
| **Pattern awareness** | Tags or themes (e.g. stress, decisions, wins) to surface patterns over time. Optional mood/energy field. |

## Apply to Systems Journal

- Optional 10-min reminder
- Prompt templates (planning vs reflection)
- Brain-dump entry type
- Optional mood/theme tags for pattern discovery
- Uses shared editor for content
