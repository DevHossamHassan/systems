# Systems/Experiences Feed

## Overview

Searchable repository of reusable guides, packing lists, and checklists. Community or personal. Sorted by votes/likes.

## Visibility

| Visibility | Description | Searchable By | Use Case |
|------------|-------------|---------------|----------|
| **Personal** | Only visible to you | Only you | Private checklists, drafts, personal runbooks |
| **Unlisted** | Hidden from discovery; accessible via link only | Only you (or anyone with link) | Share with family/friends without exposing to community |
| **Public** | Discoverable in search and feed | Everyone | Community-contributed experiences, templates for others |

Per-entry control: user chooses visibility when creating/editing each system entry.

## Terminology

- **Reusable packing list** – Create once, reuse for every trip
- **Trip preparation checklist** – Chronological tasks before travel
- **Travel runbook / playbook** – Repeatable process for trip planning
- **Gear list / Loadout** – Itemized list with metadata
- **Digital garden** – Public personal knowledge; share in-progress ideas

## Examples

### Hiking Egypt

- Search "hiking egypt" → feed of packing lists
- What to take, what's needed vs what's not
- Sorted by votes/likes
- User can add entry from previous trip (what worked, what didn't)

### Expat visiting home country

- Stuff to buy (gifts, local products)
- What to bring from home
- What to bring back
- Reuse and refine across visits

### Trip booking preparation

- 6+ months: passport, visa, entry approvals
- 3–4 months: flights, accommodation, insurance
- 1–2 months: confirmations, vaccinations
- 1–2 weeks: weather, offline maps
- 24–48h: flight confirmations, transport

## Features

- Search UX (query → results feed)
- Entry structure: MD, editor, well-organized
- Votes/likes and sorting
- Copy to my systems (fork public → personal)
- Categories: packing, trip prep, runbook, etc.
- What worked vs didn't (metadata)
- Built-in templates by trip type
- Shakedown/feedback (community comments on shared entries)
