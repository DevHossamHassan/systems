# Planned Features

## Systems/Experiences Feed

- Search (e.g. "hiking egypt")
- Feed/list of entries
- Votes/likes sorting
- Community + personal entries
- Visibility: Personal / Unlisted / Public per entry
- Copy to my systems (fork public → personal)
- Categories: packing, trip prep, runbook, etc.
- What worked vs didn't (metadata)
- Built-in templates
- Shakedown/feedback (community comments)

## Notes

- Dedicated notes tab using shared editor
- Robust search (Meilisearch)
- Templates by note type
- Tags and links
- Optional weekly review nudges
- Capture rules guidance in onboarding
- See [notes-feature.md](notes-feature.md)

## Journal

- Room persistence (or equivalent for web)
- ViewModel, use cases, repository
- 10-min reminder, prompt templates
- Morning (planning) vs evening (reflection)
- Brain dump mode
- Mood/theme tags for pattern discovery
- See [journal-feature.md](journal-feature.md)

## People

- Conversation notes (build on next meeting)
- Birthday logging + optional BD notifications (per person)
- Work contacts: title, background, notes from last convo
- Editor integration for all people notes
- Activity timeline per person
- Reconnection reminders
- See [people-feature.md](people-feature.md)

## Media

- Dedicated tab
- Categorized, sorted, searchable
- Media picker for editor

## Editor (Web Enhancements)

- Live preview (Confluence/MarkText style)
- Full MD support
- Modular reuse across features

## Notifications

- Birthday reminders (People)
- Other reminders

## Settings

- App configuration
