# People Feature

## Overview

Log things about people you know or meet. Build on conversations over time. Track birthdays and work context.

## Core Use Cases

### Conversation notes

- Note what you discussed
- Build on it when you meet again
- Avoid "what did we talk about last time?"

### Birthday logging

- Store birthday per person
- Optional BD notifications (per person, on/off)
- Days-before reminder (e.g. 2 weeks for gift shopping)

### Work contacts

- Title, company, role
- Background
- Notes from last conversation
- Editor used for all notes about people

## Features

- Add person
- Activity timeline per person (conversations, BDs, notes with timestamps)
- Structured fields: role, company, last_convo_date, last_convo_notes
- Editor integration for rich notes
- Optional reconnection reminders (e.g. "Haven't contacted in 30 days")
- BD notifications via job queue (email/push)
